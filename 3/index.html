Overview

In this project I implemented core filtering operations (convolution, finite differences, Gaussian & DoG), explored frequency-domain intuition via unsharp masking (image sharpening), created hybrid images (Oliva et al., SIGGRAPH 2006), and built Gaussian/Laplacian stacks to perform multi-resolution blending (Burt & Adelson, 1983). Below are results, brief methods, and observations for each subsection.
