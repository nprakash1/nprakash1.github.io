Project 2 — Fun with Filters and Frequencies

Overview

This project explores spatial filtering and frequency-domain intuition. I re-implemented convolutions, built derivative and Gaussian filters, created hybrid images by combining low/high frequencies, and performed multi-resolution blending via Gaussian/Laplacian stacks. Below are concise write-ups and results for each part.

Part 1 — Fun with Filters

1.1 Convolutions from Scratch

I implemented 2D convolution in NumPy with padding (four-loop and two-loop variants) and compared results/time with scipy.signal.convolve2d. Below is a single summary panel that includes my grayscale selfie, 9×9 box filter result, and Dx/Dy finite-difference outputs plus a runtime note.

Takeaways

High-frequency control explains “sharpness,” while pyramids/stacks let us manipulate bands for compelling composites. Alignment and mask smoothness matter as much as parameter choices. Visualizing spectra and per-level contributions makes the results predictable.
